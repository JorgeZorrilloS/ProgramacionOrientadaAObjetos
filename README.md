# Tareas de Programación en C++

Este repositorio contiene diferentes programas desarrollados en C++ como parte de prácticas y tareas de Programación Orientada a Objetos (POO).

Los ejercicios fueron creados para practicar conceptos fundamentales como clases, objetos, métodos, encapsulamiento e interacción con el usuario mediante consola.

---

# Tecnologías utilizadas

- Lenguaje: C++
- Paradigma: Programación Orientada a Objetos
- Compilación en consola

---

# Programas incluidos

## Piedra, Papel o Tijera

Implementación del juego clásico Piedra, Papel o Tijera donde el usuario juega contra la computadora.

Características:

- Selección de jugada del usuario
- Generación de jugada aleatoria para la computadora
- Comparación de resultados
- Mensaje indicando ganador, perdedor o empate

Conceptos aplicados:

- Clases
- Condicionales
- Lógica de comparación

---

## Sistema de Autobús

Programa que simula el funcionamiento básico de un autobús.

Características:

- Control de pasajeros
- Manejo de la capacidad del autobús
- Visualización del estado del sistema
- Interacción con el usuario en consola

Conceptos aplicados:

- Objetos
- Métodos
- Manejo de estados

---

## Pieza Torre (Ajedrez)

Simulación del movimiento de una torre de ajedrez en un tablero de 8x8.

Características:

- Representación de torres blancas y negras
- Validación de movimientos horizontales y verticales
- Verificación de posiciones dentro del tablero
- Visualización del estado actual de las piezas

Conceptos aplicados:

- Encapsulamiento
- Validación de lógica de movimiento
- Métodos para control de posición
